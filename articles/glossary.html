Глоссарий

Алгоритм Конечная последовательность точных действий, формальное выполнение которых позволяет получить решение задачи для любого допустимого набора исходных данных.
Алгоритмическая конструкция ветвление Обеспечивает выполнение одной или другой последовательности команд в зависимости от истинности или ложности некоторого условия.
Алгоритмическая конструкция повторение (цикл) Представляет собой последовательность действий, выполняемых многократно.
Анимация (от лат. animare — оживить) Процесс изменения размера, положения, цвета или формы объекта с течением времени.
Анимация движения Анимация, в которой достаточно задать состояние объекта в начале и в конце движения, а все промежуточные фазы движения рассчитает программа.
Анимация формы Плавное изменение объекта анимации.
Аудиоинформация (звукозапись) Звуковая информация, записанная каким-либо образом, пригодным для воспроизведения.
Аудиоредактор Программное средство, которое используется для редактирования аудиофайлов.
Аудиофайл Файл с аудиоинформацией в цифровой форме
Базовые алгоритмические конструкции Следование, цикл и ветвление.
Битрейт Количество бит двоичной записи, которое приходится на секунду воспроизведения (скорость потока).
Видеоконвертер Программное средство для конвертации видеофайлов.
Видеопереход между клипами Эффект смены, который можно вставлять между клипами.
Видеоредактор Программное средство для видеомонтажа.
Видеоряд Полоса из условных изображений видеофрагментов, которая отражает структуру видеофильма.
Вспомогательный алгоритм Алгоритм, который можно использовать в других алгоритмах, указав его имя и, если необходимо, значения параметров.
Высказывание Повествовательное предложение (утверждение), о котором можно сказать, истинно оно или ложно.
Звукозапись (аудиоинформация) Звуковая информация, записанная каким-либо образом, пригодным для воспроизведения.
Исполнитель Человек, группа людей или техническое устройство, которые способны правильно выполнять команды алгоритмов.
Кадры Изображения последовательных фаз движения объектов или их частей.
Клип Видеофрагмент.
Ключевые кадры Кадры, содержимое которых определяется автором анимации.
Кодек Специальная программа, которая сжимает (уменьшает) и восстанавливает первоначальный объем аудио- или видеофайла.
Колонтитул Текст или изображение, которые размещаются на краю каждой страницы документа и повторяются на всех страницах.
Команды управления (управляющие конструкции) Команды, которые управляют порядком выполнения других команд в программе (команды цикла и ветвления).
Компьютерная анимация Создание анимации с помощью компьютера.
Компьютерный видеомонтаж Процесс создания видеофильма из видеофрагментов с помощью специального программного средства.
Конвертация аудиофайла (видеофайла) Процесс изменения его формата.
Логические операции НЕ, И, ИЛИ, в PascalABC not, and, or (соответственно).
Логический тип данных boolean Тип данных, предназначенный для работы с условиями. Величины типа boolean могут принимать два значения — false (ложь) и true (истина).
Логическое выражение Выражение, принимающее одно из двух значений: true или false.
Маркированный список Список абзацев, отмеченных маркерами.
Медиаплеер Плеер, который воспроизводит как звук, так и видео.
Многоуровневый список Список абзацев, в которых применяются одновременно различные виды номеров и/или маркеров.
Направляющий слой Слой, содержащий траекторию движения.
Нумерованный список Список абзацев, отмеченных номерами.
Оглавление Указатель заголовков в документе, отражающий его структуру и ускоряющий поиск частей произведения по номерам страниц.
Оператор ветвления Команда, реализующая алгоритмическую конструкцию ветвление на языке программирования.
Оператор присваивания Предназначен для того, чтобы задавать значения переменным и вычислять значение выражений.
Оператор цикла Команда, реализующая алгоритмическую конструкцию повторение на языке программирования.
Плеер Программа для воспроизведения аудио- и видеофайлов.
Программа Последовательность команд, исполнителем которой является компьютер. Представляет собой запись на некотором формальном языке — языке программирования.
Публикация Сохранение фильма в формате SWF.
Редактирование аудиофайла Процесс его изменения, который заключается в вырезании, вставке, удалении и комбинировании частей аудиофайла, которые называются фрагментами.
Символ (флэш) Объект, помещенный в библиотеку.
Система проверки правописания Компьютерная программа, осуществляющая проверку заданного текста на наличие в нем орфографических ошибок.
Следование Последовательность команд алгоритма, которые выполняются в том порядке, в котором они записаны. Среди команд, образующих алгоритмическую конструкцию следование, отсутствуют команды, меняющие порядок выполнения других команд.
Составное условие Логическое выражение, в котором наряду с простыми условиями (сравнениями) используются логические операции.
Список Абзацы, отмеченные маркерами или номерами.
Стиль Набор параметров форматирования текста.
Текстовый клип Видеофрагмент с текстовой надписью на прозрачном или цветном фоне.
Условие для исполнителя Известное ему высказывание, которое может соблюдаться (быть истинным) либо не соблюдаться (быть ложным).
Фильм Документ, созданный в Flash.
Цикл с параметром Используется тогда, когда известно количество повторений.
Цикл с предусловием Используется в том случае, когда известно условие продолжения работы.
Экземпляр Копия символа, помещенная в рабочую область.

Гласарый

Абзац Тэкст, які заканчваецца знакам абзаца: ¶.
Адбіццё відарыса Люстраное адлюстраванне відарыса адносна нябачнай восі (гарызантальнай або вертыкальнай).
Алгарытм Зразумелая і канечная паслядоўнасць дакладных дзеянняў (каманд), фармальнае выкананне якіх дазваляе атрымаць рашэнне пастаўленай задачы.
Браўзер Праграма, прызначаная для прагляду вэб-старонак.
Буфер абмену Спецыяльная вобласць памяці камп’ютара для часовага захоўвання даных пры выкананні аперацый Вырезать і Копировать.
Выканаўца алгарытму Чалавек, група людзей або тэхнічнае ўстройства, якія разумеюць каманды алгарытму і ўмеюць правільна іх выконваць.
Вэб-сайт Некалькі вэб-старонак з агульнай тэмай і зместам, звязаныя паміж сабой гіперспасылкамі.
Вэб-старонка Дакумент або інфармацыйны рэсурс Інтэрнэту.
Графічны рэдактар Спецыяльная праграма, прызначаная для стварэння і рэдагавання відарысаў.
Графічны спосаб запісу алгарытму Запіс алгарытму з дапамогай геаметрычных фігур (блокаў), якія адпавядаюць камандам алгарытму, і ліній для злучэння блокаў.
Даныя Звесткі, прыведзеныя ў фармалізаваным выглядзе (на адной з фармальных моў).
Дапаможны алгарытм Алгарытм, які можна цалкам выкарыстоўваць у іншых алгарытмах.
Інтэрнэт Сусветная камп’ютарная сетка, здольная перадаваць даныя з аднаго пункта зямнога шара ў любы іншы.
Інфарматыка Навука, якая вывучае метады ўяўлення, захоўвання, перадачы, апрацоўкі і выкарыстання інфармацыі з ужываннем камп’ютарных тэхналогій у розных сферах чалавечай дзейнасці.
Інфармацыя Звесткі пра аб’екты і з’явы навакольнага свету.
Каманда ў алгарытме Накіраванне на выкананне канкрэтнага дзеяння.
Камп’ютар Універсальнае ўстройства для работы з данымі.
Камп’ютарная прэзентацыя Электронны дакумент, створаны на камп’ютары з дапамогай спецыяльных праграм і прызначаны для дэманстрацыі гледачам.
Камп’ютарныя тэхналогіі Тэхналогіі работы з данымі, заснаваныя на выкарыстанні камп’ютараў.
Праграмнае забеспячэнне (ПЗ) Сукупнасць усіх праграм, напісаных чалавекам для камп’ютара.
Праграмны спосаб запісу алгарытму Запіс алгарытму ў выглядзе праграмы.
Прэзентацыя Спосаб уяўлення інфармацыі з дапамогай тэхнічных сродкаў або без іх, прызначаны для дэманстрацыі чаго-небудзь новага (праекта, тавару і г. д.).
Рэдагаванне Змяненне відарыса або зместу тэксту.
Сімвал тэксту Кожная літара, лічба, знак прыпынку і г. д.
Слайд Асобная старонка прэзентацыі, прызначаная для змяшчэння матэрыялаў.
Слоўны спосаб запісу алгарытму Запіс алгарытму на натуральнай мове зносін.
Тэкставы дакумент Дакумент, інфармацыя ў якім пададзена ў выглядзе тэксту.
Тэкставы курсор Паказальнік месца ўводу сімвала.
Тэкставы рэдактар Праграма, прызначаная для работы са змесцівам электронных тэкставых дакументаў.
Фармаціраванне тэксту Змяненне формы ўяўлення тэксту.
Фрагмент відарыса Вылучаная частка відарыса.
Фрагмент тэксту Вылучаная частка камп’ютарнага тэксту.
Электронная пошта Служба сеткі Інтэрнэт, прызначаная для абмену паведамленнямі (электроннымі лістамі) з дапамогай электронных паштовых скрынь.
Электронны тэкставы дакумент Тэкставы дакумент, захаваны на дыску ў файле.
